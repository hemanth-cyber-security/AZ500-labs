# 🔐 Azure Security Labs – AZ-500

This repository contains hands-on labs completed as part of the official  
**AZ-500: Microsoft Azure Security Technologies** learning path.

🔗 Lab Source: https://microsoftlearning.github.io/AZ500-AzureSecurityTechnologies/

---

## 🎯 Purpose

The purpose of this repository is to:

- Practice Azure security concepts through hands-on labs  
- Gain practical experience with Azure security services  
- Build a structured learning path toward cloud security roles  
- Prepare for the AZ-500 certification  

---

## 📘 About AZ-500

The AZ-500 certification focuses on implementing and managing security in Azure, including:

- Identity and Access Management  
- Platform Protection  
- Data and Application Security  
- Security Operations  

These labs provide guided exercises aligned with real-world Azure security scenarios.
